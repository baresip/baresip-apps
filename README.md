# baresip-apps
Baresip Applications

## Building

See [Wiki: Install Stable Release](https://github.com/baresip/baresip/wiki/Install:-Stable-Release)
or [Wiki: Install GIT Version](https://github.com/baresip/baresip/wiki/Install:-GIT-Version)
for a full guide.

```bash
$ cd baresip
$ cmake -B build
$ cmake --build build
$ cmake --install build
```
